Site de referência para o CID-10

http://tabnet.datasus.gov.br/cgi/sih/mxcid10lm.htm

COMO PROCEDER

  • Abra o aplicativo Microsoft Excel
  • Clique na ABA Dados
  • Clique no botão Da Web
  • Na barra de endereços digite ou cole o endereço do site que conté a tabela do Cid10.
  • Procure na página um botão amarelo com seta preta apontando para a direita.
  • Apos selecionar a tabela, clique em importar
  • Depois que importar os dados para o excel, salve o arquivo no formato CSV (separado por vírgulas)

    •  

    Atenção:

    Ante de importar a tabela do CID 10 verifique se a tabela consta com as seguintes colulas, se os nomes das colunas estiverem diferentes, altere-os para que fique igual ao exemplo abaixo:

    Codigo | Descricao | Codigos_da_CID_10

    Caso haja mais colunas, você deverá excluí-las antes de realizar a importação da tabela.
